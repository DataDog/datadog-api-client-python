# TagToHosts

In this object, the key is the tag, the value is a list of host names that are reporting that tag.

| Name     | Type                | Description                          | Notes      |
| -------- | ------------------- | ------------------------------------ | ---------- |
| **tags** | **{str: ([str],)}** | A list of tags to apply to the host. | [optional] |
